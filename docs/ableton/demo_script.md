# 90-second demo script

*Target: one continuous screen recording of the Streamlit app, built-in demo
session, ~90 seconds. Timings are guidance, not hard cuts.*

---

**[0:00–0:12] Opening problem**

> "DAW sessions contain rich production knowledge, but most AI music systems
> only see audio, text, or isolated parameters. The session itself — tracks,
> routing, device chains — stays invisible."

*On screen: app header and mode selector.*

**[0:12–0:24] Prototype introduction**

> "Ableton Session State Explorer represents an Ableton-style session as an
> interpretable graph. Here's a built-in indie vocal production sketch: six
> tracks, three scenes, twenty-two devices, two return tracks, and a master
> chain."

*On screen: load the demo session; show the session summary metrics.*

**[0:24–0:42] Walkthrough**

> "Tracks, clips, scenes, devices, sends, returns, and the master chain become
> nodes and edges. Every element is typed, labeled, and inspectable — and the
> view can be filtered down to the production structure alone."

*On screen: interact with the graph — hover a device node, toggle
"Production structure only", focus on the Lead Vocal track.*

**[0:42–0:56] Analysis**

> "Uploaded stems or mixdowns are analyzed with audio descriptors — RMS, peak,
> spectral shape, tempo, loudness — linking session structure to acoustic
> outcome."

*On screen: upload a stem, assign it to a track, show the descriptors table.*

**[0:56–1:16] Recommendation**

> "The system produces graph-level suggestions with explanations and caveats —
> such as return tracks that are defined but never used, or a vocal chain with
> no corrective stage. Each suggestion cites the exact graph nodes it reasons
> about, and each one is phrased as an option, not an instruction."

*On screen: scroll the recommendations section; expand the unused-returns and
vocal-chain cards.*

**[1:16–1:30] Research value**

> "Everything exports as transparent JSON — session state, graph, descriptors,
> recommendations. This is a small prototype for studying DAW-state
> representation, prediction, and explanation in human-centered AI-assisted
> production."

*On screen: export section; click "Complete bundle JSON"; end on the research
framing box.*
